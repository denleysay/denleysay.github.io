运行环境

本地

使用jupyter notebook，这里采用的是docker容器

  1. 在安装了docker-compose的终端执行：docker-compose up
  2. 在浏览器中打开日志中最后打印出的网址，如 http://localhost:8888/?token=1c267204f7e6e244f39ab866d7c25ac4ef3e7aecc1bc7ddc
  3. 按CTRL + C停止运行

详见：TensorFlow示例中的Dockerfile和docker-compose.yml

远程

使用Google Drive的Colaboratory

  • 使用方法: 薅资本主义羊毛，用谷歌免费GPU
  • 上传数据: 完全云端运行：使用谷歌CoLaboratory训练神经网络
  • 两种访问google drive的方式
  1. Here’s an example of using a FUSE Drive interface to access your Drive files like local files
  2. I’m guessing you also found the bundled example I/O notebook, which shows how to use Python APIs to access files as well.

方法

  1. 看普通程序员如何转向AI方向—关注他的博客
  2. TensorFlow 教程
  3. 吴恩达的机器学习，不是cs229的版本
  4. 并不一定需要完全掌握数学，也可以提前开始做一些尝试和学习
  5. 啃基础，强烈建议看台湾大学李宏毅的机器学习和深度学习:
  6. 参加数据科学竞赛
  • 一个进行数据发掘和预测竞赛的在线平台kaggle
  • 从零开始，教初学者如何征战全球最大机器学习竞赛社区Kaggle竞赛
  • ImageNet/WebVision
  • google机器学习速成课程(含术语表)
    • 谷歌发布机器学习速成课，居然是中文“授课”
  • 从机器学习谈起了解大概（入门）
    1. 线性代数：矩阵乘法；
    2. 高数：求导；
    3. 概率论：条件与后验概率
  • 使用google的models模型实践中搞懂流程
  • 将训练好的模型加入到你的app中，docker容器中，云端服务器中
